# Personal Finance Transaction Organizer

PF is a full-stack web application that leverages generative AI to allow users to accurately categorize and gain valuable insights from their transaction data. The app includes a modern front-end built with React and a custom back-end Python API server. This README documents how to install, run, and contribute to the project. It also explains the motivation, architecture, and structure after merging the two repos.

## Features
Client-side UI built with React — modular components, state management, routing, responsive design, etc.

Back-end API server for data provision — supports CRUD operations, authentication/authorization, data persistence, etc.


# Tech Stack

Front-end: HTML, CSS React.js, Typescript 
Back-end: Python(Flask), OpenAI API
Communication: RESTful HTTP API 
Data store: MongoDB
Package management: npm

Easy local setup and configuration for both client and server.
